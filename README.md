###### 2. 구현 기능 정리
- **게시판 기능**
    - 모든 게시글, 특정 카테고리 게시글 조회
    - 게시글 검색(제목, 작성자, 내용)
    - 게시글 상세보기
    - 게시글 작성
    - 게시글 수정
    - 게시글 삭제
    - 게시글 답글 작성
    - 게시글 추천하기 
    - 게시글 댓글개수 순, 추천개수 순 정렬
- **회원기능**
    - 회원가입
    - 로그인
    - 회원탈퇴(작성게시글, 작성 댓글 삭제)
    - JWT 기반 사용자 인증인가
- ** 번외기능**
    - 전체 회원 수 (등급별 조회)
    - Swagger 3.0 기능 도입
